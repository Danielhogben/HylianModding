About

What is HylianModding?

An autonomous game development studio combining AI-powered tooling with traditional reverse engineering to create mods for classic games and original titles.

🤖

AI-Powered

AI Dungeon Master engine that generates dynamic gameplay, NPCs, and world events using local LLMs.

🔧

Reverse Engineering

Deep N64 ROM analysis with Ghidra, custom disassembly pipelines, and asset extraction tools.

🌐

Distributed

Docker Swarm orchestration across Nexus GPU server, Jupiter media hub, and Luna edge node.

🎨

Open Source

MIT licensed. All tools, pipelines, and game mods are open for the community to use and extend.

Getting Started

Quick Start

01

Clone

git clone https://github.com/Danielhogben/HylianModding.git
cd HylianModding
02

Setup

python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
03

Explore

ls AI_DM/        # AI Engine
ls CoOp/         # Co-op framework
ls ROM_Hacks/    # ROM projects
ls Tools/        # Dev tools
04

Contribute

git checkout -b feature/my-feature
# Make changes, test, commit, PR
